# 🐾 Charme - Pet in Fashion

Bem-vindo ao repositório do site da **Charme**, minha loja de roupinhas pet feitas à mão. Este projeto nasceu do desejo de unir minha paixão pelo artesanato com o aprendizado de desenvolvimento web.

> 🧶 **Sobre a Charme:** Todas as peças são produzidas artesanalmente por mim, com foco em conforto e estilo para os pets.

---

## 👩‍💻 Objetivo do Projeto
Este site está sendo construído do zero com dois propósitos principais:
1. **Prática de HTML5 Semântico:** Aplicar as tags corretas para garantir acessibilidade e uma boa estrutura de dados.
2. **Aprendizado de CSS3:** Explorar a estilização de elementos, posicionamento e design visual.

## 🛠️ Tecnologias em Estudo
* **HTML5:** Estruturação semântica (header, nav, main, section, footer).
* **CSS3:** Cores, tipografia, bordas e as primeiras noções de layout.
* **Git & GitHub:** Controle de versão para acompanhar a evolução do meu código.

## 📂 O que você vai encontrar aqui
* `index.html`: A vitrine principal com as novidades.
* `peças.html`: Catálogo das roupinhas que eu mesma produzo.
* `medida.html`: Guia de tamanhos para ajudar os tutores a escolherem o tamanho certo.
* `fale_conosco.html`: Formas de entrar em contato para pedidos.
* `style.css`: Toda a "magia" visual que estou aprendendo a criar.

## 🚀 Status
O projeto é um **estudo contínuo**. À medida que aprendo novos conceitos de CSS ou HTML, volto aqui para refinar o código e deixar o site da Charme ainda mais profissional.
